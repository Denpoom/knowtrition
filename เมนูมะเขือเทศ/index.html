มะเขือเทศผัดไข่

วัตถุดิบ

มะเขือเทศ / ไข่ไก่ / ต้นหอม / น้ำมันพืช / เกลือ / กระเทียมสับ / น้ำตาล


ขั้นตอนการทำ

1. ล้างมะเขือเทศให้สะอาด แล้วทำการลอกเปลือกออกโดยการลวกด้วยน้ำร้อน เคล็ดลับ: ใช้มีดกรีดที่ผิวเล็กน้อยเพื่อให้ลอกเปลือกได้ง่าย

2. หั่นมะเขือเทศที่ลอกเปลือกแล้วเป็นชิ้นขนาดพอดีคำ ตีไข่สำหรับเจียว พร้อมหั่นต้นหอมเป็นท่อน และสับกระเทียมเตรียมได้สำหรับผัดในขั้นตอนต่อไป เคล็ดลับ: เนื้อมะเขือเทศสับจะช่วยให้เวลาผัดจะช่วยเพิ่มน้ำซอส และความเข้มข้นของอาหารมากขึ้น ซึ่งเนื้อมะเขือเทศสับ จะมีหรือไม่มีก็ได้ แล้วแต่ความชอบ

3. เจียวไข่ให้เหลืองสวย แล้วตักขึ้นพักไว้ เตรียมผัดในขึ้นตอนต่อไป

4. ผัดมะเขือเทศที่เตรียมไว้ ให้สุกและคายน้ำซอส อาจใส่น้ำต้มสุกนิดหน่อย จากนั้นใส่ต้นหอม ผัดให้หอม

5. นำไข่เจียวที่เตรียมไว้ ใส่ลงผัดรวมกับมะเขือเทศที่ผัดได้ที ผัดให้เข้ากัน จนน้ำซอสคลุกเคล้าทั่วไข่เจียวที่ใส่ ปรุงรสด้วย น้ำตาลเล็กน้อยตัดความเปรี้ยวของมะเขือเทศ จากนั้นใส่เกลือตามความชอบ และใส่กระเทียมสับ ผัดให้หอม แล้วตักเสิร์ฟได้

ซุปมะเขือเทศ

วัตถุดิบ

มะเขือเทศ / น้ำมันมะกอก / กระเทียม / ไทม์ / น้ำสต๊อกไก่ / เกลือป่น / พริกไทยป่น / เนยสด


ขั้นตอนการทำ

1. ตั้งเตาอบอุณหภูมิ 200 องศาเซลเซียส โรยน้ำมันมะกอกบนกระเทียมเล็กน้อยแล้วห่อด้วยฟอยล์ หั่นมะเขือเทศตามขวางวางหงายเรียงบนถาดอบโรยเกลือป่น พริกไทยป่น และน้ำมันมะกอก วางห่อกระเทียมบนถาด แล้วนำเข้าอบจนเหลือง ประมาณ 30-45 นาที พักไว้ให้เย็น

2. ปั่นมะเขือเทศและกระเทียมให้ละเอียด

3. เทมะเขือเทศใส่หม้อน้ำสต๊อกไก่ ใส่ไทม์ แล้วตั้งไฟจนเดือดลดไฟลงแล้วเคี่ยว 20 นาที ยกออกจากเตา ปรุงรสด้วย เกลือป่น พริกไทยป่น
