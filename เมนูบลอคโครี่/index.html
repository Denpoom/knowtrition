บล็อกโคลีอบชีส

วัตถุดิบ

บร็อคโคลี่หั่นพอคำต้มสุก / เนยสด / หอมใหญ่สับละเอียด / แป้งสาลี / นมสด / น้ำตาลทราย / เกลือ / พริกไทย / พาเมซานชีส / มอสซาเรลล่าชีส


ขั้นตอนการทำ

1. ทำครีมโดยใส่เนยสด นำหอมลงผัดพอเหลือง

2. นำแป้งสาลีลงผัดใส่นม

3. ปรุงรสด้วย เกลือ น้ำตาล พริกไทย

4. เคี่ยวจนเดือดข้น ใส่พาเมซานชีส

5. นำบร็อคโคลี่ลงไปผัดคลุก

6. ปาดเนยบางๆ ลงบนภาชนะที่จะนำเข้าเตาอบ

7. นำบร็อคโคลี่ที่ผัดใส่ลงภาชนะที่จะนำเข้าเตาอบ

8. โรยหน้าก่อนอบด้วย พาเมซานชีส และ ชีสมอสซาเรลล่า

9. นำไปอบในเตา ที่อุณหภูมิสูงสุด ไฟบน

10. จนชีสละลายเยิ้ม มีรอยเหลืองไหม้ พอสวยงาม จัดเสิร์ฟ

บล็อกโคลีผัดกุ้ง

วัตถุดิบ

บล็อคโคลี่ / กุ้งสด / แครอทหั่นแว่น / กระเทียมสับ / พริกไทยป่น / น้ำตาลทราย / ซีอิ้วขาว / น้ำเปล่า / น้ำมันหอย / น้ำมันสำหรับผัด / เกลือป่น


ขั้นตอนการทำ

1. เลือกบล็อคโคลี่ที่สด สีเขียวไม่แก่ จากนั้นนำมาล้างน้ำให้สะอาด สะเด็ดน้ำพักไว้

2. จากนั้นนำบล็อคโคลี่ มาตัดเอาโคนแข็งทิ้งไป แล้วหั่นเป็นชิ้นๆ ขนาดพอคำ จากนั้นปอกเปลือกแครอท ล้างน้ำให้สะอาด แล้วหั่นเป็นแว่นๆ เตรียมไว้

3. จากนั้นตั้งหม้อใส่น้ำลงไป ใช้ไฟแรง รอให้น้ำเดือดพล่าน จากนั้นใส่เกลือป่นลงไป จากนั้นนำบล็อคโคลี่หั่นชิ้น และแครอทหั่นแว่นใส่ลงไปลวก ใช้เวลาลวกประมาณ 2 นาทีพอสุก จากนั้นจึงตักแช่น้ำเย็นสักครู่ แล้วจึงตักสะเด็ดน้ำพักไว้

4. เลือกกุ้งสดที่สดใหม่ นำมาปอกเปลือกออก แล้วผ่าหลังดึงเส้นดำออก จากนั้นล้างน้ำให้สะอาด สะเด็ดน้ำพักไว้

5. จากนั้นตั้งกะทะให้ร้อน ใช้ไฟกลาง จากนั้นใส่น้ำมันสำหรับผัดลงไป รอให้น้ำมันร้อน จากนั้นจึงใส่กระเทียมสับลงไป ผัดคลุกเคล้าให้หอม จากนั้นจึงนำกุ้งสดใส่ลงไป ผัดคลุกเคล้าให้กุ้งพอสุก

6. จากนั้นจึงใส่บล็อคโคลี่ และแครอทลวกพอสุกลงไป ตามด้วยใส่เครื่องปรุงรสต่างๆ ลงไป ใส่น้ำมันหอย น้ำตาลทราย ซีอิ้วขาว น้ำเปล่า พริกไทยป่น ลงไป

7. จากนั้นผัดคนคลุกเคล้า ให้ส่วนผสมทุกอย่างเข้ากันดี จากนั้นปิดไฟยกลง จะได้กุ้งผัดบล็อคโคลี่หอมๆ พร้อมจัดเสิร์ฟ
